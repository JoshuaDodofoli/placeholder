# Placeholder

> The space between thought and completion.

Placeholder is a creative space designed to give unfinished work a home. We use AI-generated images to rapidly explore, test, and iterate on ideas—allowing us to build without overthinking and evolve without permission. 

## About the Project
This project serves as an interactive portfolio and sandbox for visual concepts including:
- **Photography** (e.g., Dusk & Grain)
- **Illustration** (e.g., Scrawl)
- **Logo Design** (e.g., Ova Studio)
- **Graphic Design** (e.g., Terrain)
